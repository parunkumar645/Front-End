## 3ï¸âƒ£ HTML Document Structure (Boilerplate)

The HTML boilerplate is the basic skeleton or starting template of any HTML document. It includes essential tags needed for every webpage.

### Standard HTML5 Boilerplate Code:

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document Title</title>
</head>
<body>
    <!-- Your content goes here -->
</body>
</html>
```
